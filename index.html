Основные уточнения перед прочтением.

  • Незнание правил сервера не освобождает вас от ответственности.
  • Заходя на наш сервер, вы автоматически соглашаетесь со всеми правилами, перечисленными в данном документе.
  • Этот документ является единственными официальными правилами на данном сервере.
  • В случае, если вы считаете администрацию неправой, то можете обратиться к Старшему Администратору и выше.
  • Свидетели могут быть использованы по желанию курирующего администратора.
  • Администрация сервера имеет право додумывать правило для разбора жалобы.
  • Старшая администрация имеет право банить без жалоб в рп профе.
  • И также старшая администрация может вас забанить по своему усмотрению.
  • При указании бана значение 0 = permanent.
  • При выдаче бана надо писать не пункт а правило.
    • Основные термины.
    • 1.1 NonRP - Неправильная отыгровка/абсолютная неотыгровка своей роли. [Наказание - 30mi]
    • 1.2 RDM - убийство человека без RP причины. [Наказание - 30mi]
    • 1.3 MRDM - убийство 3+ человек без RP причины. [Наказание - 6h]
    • 1.4 MG - использование/передача RP информации по средствам NonRP связи. [Наказание - 30mi]
    • 1.5 NLR - при смерти вы должны забыть о всех факторах, связанных с вашей смертью. Запрещено возвращаться на место смерти пока не пройдет 3-ёх минут.
    • 1.6 PA - использование пропов в качестве лестниц, использовать пропы для помехи игре.
